# Advanced BMI Calculator (Streamlit)

## Objective

The objective of this project is to develop an interactive Body Mass Index (BMI) calculator using Python and Streamlit. The application allows users to calculate BMI based on weight and height, classify health categories, store user data, and analyze BMI trends using graphs.

---

## Project Description

This project is a web-based BMI calculator built with the Streamlit framework. Users can enter their name, weight, and height to calculate their BMI instantly. The system stores the records and allows users to view BMI history and analyze trends through graphical visualization.

---

## Steps Performed

1. Implemented BMI calculation using the standard formula  
2. Built an interactive UI using Streamlit  
3. Collected user inputs such as name, weight, and height  
4. Categorized BMI values into health categories  
5. Stored BMI records in a CSV file  
6. Displayed historical BMI data in table format  
7. Implemented BMI trend visualization using Matplotlib  
8. Calculated overall BMI statistics

---

## Tools and Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- CSV File Storage

---

## Features

- BMI calculation
- Health category classification
- Data storage for multiple users
- BMI history table
- Individual BMI trend graph
- Overall BMI statistics
